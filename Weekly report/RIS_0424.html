I test the solve function by perks.Y and perks.YDOT. In details, at each call of FEX, we have to give an array of y as input to FEX. At this time, I find an array from perks.Y, which is the most similar to y (minimum L1 norm). Then, in perks.YDOT, I find an array YDOT with a same line index as Y. FEX return the YDOT.
